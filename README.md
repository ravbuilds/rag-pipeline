# RAG Pipeline
Implement RAG pipeline using LangChain framework.

### RAG - Retrieval Augmented Generation
RAG is the process of optimizing the output of a large language model, so it references a **knowledge base outside of its training data sources** before generating a response.
LLMs are themselves trained on large data and use large number of parameters to generate output for tasks like conversation chatbots, code etc.
RAG extends the knowledge of an LLM to a specific domaion or organization's without the need to retrain the model.
